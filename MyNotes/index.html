HTML 和 CSS 学习笔记

课程概述

在本课程中，主要学习了 HTML 和 CSS 的基础知识。
● HTML 元素标签的使用；
● 基本的 HTML 文档结构；
● CSS 如何修改 HTML 页面的样式；
● 各种 CSS 样式修改实践。

HTML 元素标签

在本课程中学到了以下元素标签：粗体、斜体、超链接、添加图片、换行等。
大多数的 HTML 元素由开始标签、内容、结束标签构成，某些元素没有结束标签，称为“VOID 标签”，比如 `img` 标签；
某些开始标签内，还包含有对应的属性值，比如超链接标签中的属性 href 的值为后面跟着的 URL。
课程结束时总结了 一些概念可以复习。

文档结构

在 P0 项目之前，学到了初步的一些结构，知其然而不知其所以然。在学习 CSS 的过程中，对 HTML 的各元素都是一个矩形方框的设定理解的更加深刻，知道了设计一个网页，可以由“大块”开始，到“小块”的具体内容结束。对网页结构的整体有了很好的了解。

CSS 如何修改样式

首先了解到，CSS 是用来修改 HTML 样式的。那么它是如何修改的呢？其选择某种元素类型（比如 h1、img、p、a 等元素），或者开发者自己设定的类名（比如 div class = "classname" 中的 classname），然后设定一些样式属性，这样对应的元素类型或自定义类，都会显示为设定的样式。详见：CSS 总结和参考

CSS 修改样式实践
